# 快速入门

欢迎使用 **TRPG Workbench**——一款面向 TRPG 创作者的 AI 辅助工作台。本文带你从零完成首次启动到开始创作的完整流程。

---

## 第一步：启动应用

打开应用后，你会看到主界面顶部导航栏和首页内容区。

顶部导航包含四个入口：

| 入口 | 用途 |
|------|------|
| 规则集 | 管理规则集，关联知识库和创作风格提示词 |
| 知识库 | 管理规则书、世界观 PDF 等参考资料 |
| 模型配置 | 配置 AI 模型供应商（LLM、Embedding、Rerank） |
| 用量观测 | 查看 API 调用记录与费用统计 |

---

## 第二步：创建工作空间

首页的"**最近工作空间**"区域显示你的所有工作空间。

首次使用时列表为空，点击 **「新建工作空间」** 按钮创建第一个工作空间。

已有工作空间时，每张卡片提供三个操作：
- **打开** — 进入工作台
- **设置** — 修改工作空间名称和模型路由配置
- **删除** — 永久删除工作空间

---

## 第三步：配置 AI 模型

工作空间创建后，在开始创作之前，需要先配置 AI 模型。点击顶部导航的 **「模型配置」** 进入设置页。

模型配置包含四个子页面：

- **LLM 语言模型** — 添加用于生成内容的大语言模型
- **Embedding 向量模型** — 添加用于知识库检索的向量化模型
- **模型发现** — 浏览内置的 15 个 LLM 和 5 个 Embedding 模型目录
- **Rerank 重排序** — 添加用于提升检索精度的重排序模型

建议顺序：先在"模型发现"查看可用模型，再在"LLM 语言模型"和"Embedding 向量模型"中添加你持有 API Key 的模型。

---

## 下一步

- 配置好模型后，回到首页打开工作空间开始创作
- 如需导入规则书或世界观资料，前往 **「知识库」** 页面导入 PDF，再在 **「规则集」** 页面将其关联到对应规则集
- 如需了解各模型配置详情，参阅《模型配置指南》
