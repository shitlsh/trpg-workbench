# 开始创作

工作空间是你的 AI 辅助创作场所。在这里，你可以通过对话驱动 Agent 生成 NPC、场景、怪物等结构化资产。

---

## 工作台布局

打开工作空间后，进入三栏工作台：

![工作台三栏布局](/help-images/workspace.png)

| 区域 | 说明 |
|------|------|
| **左侧 — 资产列表** | 按类型分组展示所有资产（NPC、场景、怪物、地点等），支持搜索过滤 |
| **中间 — 编辑器** | 查看和编辑选中的资产内容 |
| **右侧 — Agent 面板** | 与 AI 助手对话，驱动创作流程 |

三栏均可拖拽调整宽度，也可以折叠隐藏。

---

## 使用 Agent 面板创作

### 发送创作请求

在右侧 Agent 面板的输入框中描述你的需求，按 Enter 发送（Shift+Enter 换行）。例如：

- "生成一个神秘的精灵向导 NPC，擅长占卜"
- "创建一个废弃矿井的场景，气氛阴森"
- "修改 NPC 阿尔瑟的性格，让他更多疑"

### Agent 的工作流程

1. **理解意图** — Director Agent 分析你的请求，生成执行计划（会显示意图摘要和涉及的资产类型）
2. **澄清确认** — 如果请求不够明确，Agent 会提出澄清问题，等你回答后再继续
3. **执行创作** — Agent 调用知识库检索相关规则，生成内容
4. **确认修改** — 生成完成后，Patch 确认对话框显示具体修改内容，你可以选择接受或拒绝

> Agent 会自动检索工作空间绑定的知识库，引用相关规则来指导创作。在工作流进度中可以展开查看引用来源。

对话中会出现**工具调用**折叠卡片：展示中文工具名（如「批量新建资产」「预检跨资产替换」「移动/重命名资产」等）、简要参数，以及成功时的 JSON 结果。写入类资产/Skill 成功时卡片可标有「已写入」，并在流式过程中刷新左侧资产树。

### 快捷操作

Agent 面板下方提供三个快捷按钮：

- **新建资产** — 快速发起资产创建
- **一致性检查** — 检查所有资产之间是否存在矛盾（如 NPC 引用了不存在的地点）
- **规则审查** — 检查资产是否符合知识库中的规则，给出修改建议

## 支持的资产类型

| 类型 | 说明 |
|------|------|
| 大纲 | 模组整体结构和故事大纲 |
| 场景 | 游戏中的具体场景设计 |
| NPC | 非玩家角色档案 |
| 怪物 | 怪物/敌人设计 |
| 地点 | 地理位置和环境描述 |
| 线索 | 玩家可发现的线索和信息 |
| 分支 | 剧情分支和选择 |
| 时间线 | 事件时间线 |
| 地图简报 | 地图相关信息 |
| 世界设定 | 世界观和背景设定 |

---

## 前置配置

工作台正常运行需要以下配置：

- 工作空间已绑定 **LLM Profile**（在工作空间设置中配置）
- 如需知识库检索，还需绑定 **Embedding Profile** 和对应知识库

如果 Agent 面板报错，请先检查工作空间的模型路由配置。详见 [模型配置指南](/help/model-setup)。
