---

## 打开工作空间

在首页的"最近工作空间"区域，点击工作空间卡片上的 **「打开」** 按钮，进入主工作台。

---

## 工作台布局

主工作台采用三栏布局：

| 区域 | 说明 |
|------|------|
| 左侧面板 | 资产列表（NPC、场景、怪物等结构化创作产物） |
| 中间区域 | 当前编辑的资产内容或文档视图 |
| 右侧 Agent 面板 | 与 AI Agent 交互的对话区域 |

---

## 使用 Agent 面板创作

1. 在右侧 Agent 面板的输入框中描述你的创作需求
   - 例："生成一个神秘的精灵向导 NPC，擅长占卜"
2. 点击发送或按 Enter 提交
3. Agent 根据工作空间绑定的 LLM 模型和知识库生成内容
4. 生成结果显示在 Agent 面板的对话区域

---

## 查看和保存结果

Agent 生成的内容可以：
- 直接在对话区查看
- 保存为结构化资产（如 NPC 档案、场景描述），出现在左侧资产列表
- 进一步编辑和完善

---

## 前置配置提醒

工作台正常运行需要：
- 工作空间已绑定 LLM Profile（在工作空间设置中配置）
- 如需知识库检索，还需绑定 Embedding Profile 和对应知识库

如 Agent 面板显示配置错误或无响应，请先检查工作空间的模型路由配置。
