# 知识库导入指南

知识库用于存储规则书、世界观设定等参考资料。导入后，Agent 在创作时会自动检索相关内容，让生成结果更符合你的规则体系。

知识库直属于规则集，在 [规则集](/settings/rule-sets) 页面中管理。

---

## 创建知识库

1. 进入 [规则集](/settings/rule-sets) 页面，选择一个规则集
2. 在"知识库"区域点击「新建知识库」
3. 填写名称（如"COC7 核心规则书"）、选择类型（核心规则 / 扩展规则 / 参考模组 / 怪物手册 / 世界观资料 / 房规补充）
4. 点击「创建」

创建后，新知识库出现在知识库列表中。点击知识库名称进入详情页面。

---

## 上传 PDF 文档

在知识库详情页面：

1. 点击上传区域或直接拖拽 PDF 文件到上传区
2. 系统开始自动处理

**处理阶段：**
- **文本提取** — 从 PDF 中提取纯文本
- **分块（Chunking）** — 将文档切分为可检索的片段
- **向量化（Embedding）** — 对每个片段生成向量表示

处理耗时取决于文档大小和 Embedding 模型的响应速度。处理期间可在页面查看进度。

> **前置条件：** 上传前需要先在 [模型配置](/settings/models) 中配置好 Embedding 模型，否则向量化步骤会失败。

---

## 查看导入结果

处理完成后：

- 文档出现在知识库的文档列表中，显示分块数量
- 点击文档行可展开查看摘要和 Embedding 信息
- 点击「预览」可在右侧面板浏览文档的 Chunk 内容和页面文本
- 如有质量警告（如部分页面识别率低），会在文档详情中标注

---

## 测试检索效果

在知识库详情页面，点击「检索测试」：

1. 输入查询词（如"理智值检定"）
2. 设置返回数量（top_k）和是否使用 Rerank
3. 点击「搜索」查看检索结果

每条结果显示向量相似度分数、内容预览，以及 Rerank 分数（如果启用）。这帮助你验证知识库的检索质量。

---

## 在工作空间中使用知识库

知识库直属于规则集，所有使用该规则集的工作空间会自动继承其知识库。

此外，你也可以在工作空间设置的「额外知识库」区域，为单个工作空间添加补充知识库。

两种来源的知识库会合并，Agent 在创作时会自动检索所有绑定的知识库。
