# 知识库导入指南

知识库用于存储规则书、世界观设定、NPC 档案等参考资料，支持 PDF 导入和语义检索。

> **注：** 本文档基于知识库管理页面的 DOM 状态生成。知识库导入的部分流程（进度显示、预览等）需在实际操作中体验完整细节。

---

## 进入知识库管理

点击顶部导航的 **「知识库」**，进入知识库管理页面。

页面分为两栏：
- **左侧** — 知识库列表
- **右侧** — 选中知识库的详情与文档列表

---

## 创建知识库

1. 点击页面左上角的 **「新建知识库」** 按钮
2. 填写知识库名称（例："奇幻冒险规则书"）
3. 确认创建

创建完成后，新知识库出现在左侧列表中，点击即可选中并查看详情。

---

## 上传 PDF 文档

选中知识库后，在右侧详情区：

1. 点击 **「上传文档」** 或相应的导入按钮
2. 选择本地 PDF 文件
3. 系统开始解析和向量化处理

**处理阶段：**
- 文本提取 — 从 PDF 中提取纯文本内容
- 分块（Chunking）— 将文档切分为可检索的片段
- 向量化（Embedding）— 对每个片段生成向量表示

处理耗时取决于文档大小和所配置的 Embedding 模型响应速度。处理期间可在页面查看进度。

---

## 查看导入结果

处理完成后：

- 文档出现在知识库的文档列表中
- 可查看文档摘要和分块数量
- 如有质量警告（如部分页面识别率低），会在文档详情中标注

---

## 测试检索效果

在知识库详情页，可通过检索测试框输入关键词，验证向量检索是否能正确召回相关片段，确认知识库可正常用于工作空间中的 Agent 问答。

---

## 在工作空间中使用知识库

知识库创建并导入文档后，进入对应工作空间的 **「设置」**，在模型路由配置中关联所需的知识库，Agent 即可在创作过程中检索其中的内容。
