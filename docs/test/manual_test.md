## 整体
- [ ] 帮助按钮和切换主题的按钮只在首页有，其他页面有的有帮助，有的没有，建议全局统一一下，包括每个子页面的布局目前也风格不是很统一

## Setup wizard
- ✅ Setup wizard 中rerank模型配置仅可操作跳过，多余，直接移除
- [ ] Setup wizard 中步骤不太对，配置完模型后，应当先创建规则集，再创建工作空间（因为工作空间需要绑定规则集）。

## 规则集管理
- ✅ 不需要空白内置，完全无法修改，没什么用。
- [ ] 规则集的TAG是怎么产生的，好像仅内置的恐怖调查上面有一个内置 恐怖，而且内置的规则集为什么不允许修改和删除
- [ ] 创作风格提示词无法添加自定义的提示词，选择内置的又提示Cannot modify builtin profiles
- [ ] 规则集下资产类型提示可以添加 但实际无添加等管理按钮，而且请把系统默认的 AssetType（如 NPC/Plot/Director）也展示出来，方便用户理解和使用
- ✅ 知识库需要有embedding模型才可以添加，需要添加提示和校验。
- [ ] 如果一个PDF中包含多个知识库所需的内容，当前没有考虑，可能会都按核心规则上传

## 模型配置
- ✅ 模型配置也的模型发现放到最后一个标签页
- [ ] 为什么同样是Jina AI, 在embedding和rerank里下拉配置的方式不一样，embedding里用的是OpenAI Compatible

## 工作空间设置
- ✅ 工作空间设置里还是在按知识库可以独立存在设计，有一个额外知识库，可以移除，我们目前没有独立于规则集外的知识库
- ✅ 工作空间的设置按钮放在工作空间内部更合适，并且如果没有配置完成，那么应该有告警提示用户去配置，当前工作空间页面的右上角放一个设置按钮就好
